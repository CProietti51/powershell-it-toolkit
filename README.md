# powershell-it-toolkit

Powershell automation script designed to streamline common IT support and system administration tasks

## Features

- Menu-driven interface
- Windows Update automation
- System File Checker (SFC) execution
- Application update management
- TCP/IP stack reset
- Low Disk Space/Slow Performance Fix
- Network troubleshooting

## Technologies

- PowerShell
- NetShell (netsh)
- Windows administration tools

## Purpose

This project was created to reduce repetitive manual troubleshooting tasks and improve support efficiency.
